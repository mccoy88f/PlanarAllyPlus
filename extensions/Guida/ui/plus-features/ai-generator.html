AI Generator (OpenRouter)

Pannello per testi e immagini tramite modelli collegati a OpenRouter, con attività predefinite per il tavolo (ambientazioni, PNG, mappe, import scheda, ecc.).

Configurazione

Richiede una chiave API valida e i permessi di rete sul server. Senza configurazione le funzioni AI non sono disponibili.

Testi

Genera descrizioni, dialoghi, background e contenuti strutturati; alcune attività producono JSON compatibile con la scheda personaggio o altri flussi.

Immagini

Crea immagini per token o asset e, dove previsto, inviale alla libreria senza riempire cartelle temporanee inutilmente.

MapsGen

La trasformazione “realistica” delle mappe procedurali può appoggiarsi a questo modulo quando configurato.

AI Generator (OpenRouter)

Text and image workflows via OpenRouter-backed models, with tasks tailored for play (locations, NPCs, maps, sheet import, etc.).

Setup

Requires a valid API key and server network access. Features stay disabled until configured.

Text

Generate descriptions, dialogue, backgrounds, and structured outputs; some tasks emit JSON compatible with the character sheet or other flows.

Images

Create art for tokens or assets and send results to the library where supported.

MapsGen

“Realistic” map passes can route through this stack when enabled.
