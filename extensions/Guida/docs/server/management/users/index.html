#Gestione base degli utenti del server

Informazione

Questa sezione è molto datata. Puoi fare molte operazioni con la CLI e c’è anche una vista browser di base per alcune attività amministrative.

Al momento, dalla v0.22.3, non esiste un’interfaccia grafica per la gestione degli utenti a livello server, quindi devi usare script Python per la gestione degli utenti.

Assicurati sempre di essere in una shell Python prima di eseguire uno di questi script. Apri semplicemente un terminale e digita python per aprire una shell; puoi uscire con quit()

PlanarAlly usa Peewee come strumento ORM per il database.

#Elencare tutti i nomi utente sul server

from src.db.all import User
[ u.name for u in User.select() ]

#Cambiare la password per un utente specifico

from src.db.all import User
u = User.get(name="*nome-utente*")
u.set_password("*nuova-password*")
u.save()

#Creare un nuovo utente come admin

from src.db.all import User
User.create_new("*nome-utente*", "*password*", "*email-opzionale*")
