Assets Installer

Strumento del cruscotto (dashboard) per caricare un archivio ZIP e scompattarlo nella libreria asset della campagna, rispettando la struttura di cartelle interna. È pensato per importare pacchetti di mappe, token, PNG e audio in un colpo solo.

Selezione della destinazione

Un albero cartelle mostra la libreria attuale: espandi i nodi e seleziona la cartella in cui vuoi che finiscano i file dello ZIP (ad esempio tokens/nemici o una cartella dedicata a un modulo). Se non scegli nulla, l’estensione usa una destinazione predefinita secondo il flusso dell’interfaccia.

Caricamento del pacchetto

  • Prepara un ZIP con file e sottocartelle già organizzati come desideri in libreria.
  • Avvia l’upload: compare una barra di avanzamento per file grandi.
  • Il server valida, estrae e registra i file nel database asset unificato (anteprime generate in modo asincrono).

Cosa conviene impacchettare

  • Collezioni di token PNG/WebP con nomi coerenti.
  • Mappe numerate o organizzate per avventura.
  • Strutture di cartelle replicate da bundle scaricati da comunità (rispettando licenze).

Permessi e sicurezza

Solo utenti con diritto di modifica sugli asset della campagna dovrebbero usare l’installer. Non caricare ZIP da fonti non fidate: il contenuto viene scritto nel percorso selezionato.

Dopo l’installazione

I nuovi file compaiono nel gestore asset del client di gioco; le anteprime possono comparire con breve ritardo mentre il server elabora le miniature.

Assets Installer

Dashboard tool to upload a ZIP and unpack it into the campaign asset library, preserving inner folder layout—ideal for bulk maps, tokens, and audio.

Pick a destination

A folder tree mirrors your library: expand nodes and select the target folder (e.g. tokens/enemies). If none is chosen, the UI falls back to its default target.

Upload flow

  • Build a ZIP with the hierarchy you want in-game.
  • Start upload—watch the progress bar for large archives.
  • The server validates, extracts, and registers files in the unified asset DB (thumbnails follow asynchronously).

What to pack

  • Token sheets as PNG/WebP with sensible names.
  • Numbered maps or per-adventure folders.
  • Community bundles you are licensed to use.

Permissions and safety

Only users who may edit campaign assets should run installs. Avoid untrusted ZIPs—content lands in the selected tree.

After install

New files show up in the in-game asset manager; previews may trail slightly while thumbnails generate.
