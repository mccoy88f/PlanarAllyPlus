AI Generator

Estensione che invia richieste a modelli remoti (tipicamente tramite OpenRouter, con supporto opzionale per Google AI Studio) configurati sul server. Include attività predefinite per testo e immagini, editor di task personalizzati e pannello impostazioni per chiave API e elenco modelli.

Configurazione obbligatoria

  • Il server deve avere una chiave API valida e accesso in uscita verso i provider.
  • Dalla scheda Impostazioni puoi verificare i modelli disponibili (gratuiti o meno) e salvarli.
  • Senza configurazione le esecuzioni restano disabilitate o falliscono con messaggio esplicito.

Attività (task) predefinite — esempi

  • Genera personaggio: testo descrittivo per PNG o NPC.
  • Genera JSON scheda: output strutturato compatibile con l’import della scheda personaggio.
  • Genera storia / ambientazione: narrazione, hook, descrizioni di luogo.
  • Importa scheda da file: carica un PDF o testo; il modello estrae dati verso il formato scheda.
  • Importa mappa: analisi di immagine mappa per ricavarne dati utili (secondo il prompt del task).

Puoi duplicare e modificare i prompt di sistema e utente per adattarli alla tua campagna.

Task personalizzati

Crea un task vuoto, definisci etichetta, istruzioni di sistema e prompt utente, poi salvalo nell’elenco. Utile per formati ripetitivi (incontri casuali, loot, iniziative narrative).

Immagini e multimodale

Alcuni modelli espongono modalità immagine o input vision: puoi generare arte per token, location o inviare un’immagine come riferimento. I file generati possono essere inviati alla libreria asset quando il flusso lo supporta.

Integrazione MapsGen

La trasformazione “realistica” delle mappe procedurali può chiamare la stessa infrastruttura (prompt + modello immagine) senza riempire cartelle temporanee in modo permanente.

Privacy e costi

I contenuti delle richieste transitano verso il provider scelto; controlla termini di servizio e costi per token. Non inserire segreti di campagna sensibili se non ti fidi del canale.

AI Generator

Extension that calls remote models (typically OpenRouter, optionally Google AI Studio) configured on the server. Ships with preset tasks, a custom task editor, and a settings tab for API keys and model lists.

Required setup

  • Server needs a valid API key and outbound network access.
  • The Settings tab lists available models (free/paid) for selection.
  • Without setup, runs stay disabled or error clearly.

Preset tasks (examples)

  • Generate character: prose NPC/PC descriptions.
  • Generate character JSON: structured output for the character sheet importer.
  • Generate story / setting: hooks, locations, lore blurbs.
  • Import character sheet: upload PDF/text; model extracts sheet data.
  • Import map: vision-oriented analysis of a map image (per task prompts).

Duplicate tasks to tweak system/user prompts for your table.

Custom tasks

Create blank tasks with your own instructions—great for random encounters, loot tables, or recurring formats.

Images and multimodal

Some models expose image output or vision inputs—generate token art, scenes, or pass reference images. Outputs can route to the asset library when supported.

MapsGen bridge

“Realistic” dungeon/building passes reuse this stack so schematic maps become rendered art without spamming temp folders.

Privacy and billing

Prompts leave your server for the selected provider—review their terms and token pricing. Avoid sensitive secrets if you do not trust the channel.
