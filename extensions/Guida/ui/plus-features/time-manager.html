Gestione del tempo (Time Manager)

Gestisci uno o più timer nominati in un’unica estensione: ogni blocco può funzionare come conto alla rovescia (tempo che scende) o come cronometro (tempo che sale), con avvio, pausa, reset e stato visibile anche dopo aver chiuso il modale.

Modalità per timer

  • Conto alla rovescia: imposti una durata (es. 1 minuto per round, 10 minuti pausa); quando arriva a zero può scattare un avviso sonoro.
  • Cronometro: misura quanto tempo è passato da un evento (es. durata effettiva del turno).

Più timer in parallelo

Usa il pulsante di aggiunta per creare voci separate (es. “Round 3”, “Pausa”, “Prova FOR”). Ogni riga ha nome modificabile, controlli propri e rimozione. La ricerca in barra filtra l’elenco quando ne hai molti.

Esecuzione in background

Il tempo continua a scorrere mentre passi alla mappa, alla chat o ad altre estensioni: non è necessario tenere la finestra aperta per “congelare” il tavolo.

Feedback sonoro

Alla fine di un conto alla rovescia può essere riprodotto un suono di notifica così il gruppo nota lo scadere anche se nessuno guarda il timer.

Casi d’uso

  • Limitare il tempo di decisione in combattimento.
  • Timer per prove a tempo o enigmi.
  • Pausa caffè sincronizzata tra Master e giocatori (se tutti tengono aperta la sessione).

Time Manager

Run multiple named timers: each entry can be a countdown or a stopwatch, with start/pause/reset and state that survives closing the modal.

Modes

  • Countdown: set a duration (round timer, break timer); optional audio when it hits zero.
  • Stopwatch: measure elapsed time (actual turn length, skill challenges).

Many timers at once

Add rows for separate concerns (“Round 3”, “Break”, “STR check”). Rename inline, remove when done. The toolbar search filters long lists.

Background execution

Timers keep ticking while you use the map, chat, or other extensions—you do not need the window focused to keep time fair.

Audio cue

Optional alert sound when a countdown completes so the table notices even if nobody is watching the panel.

Use cases

  • Decision timers in combat.
  • Puzzles or skill challenges on the clock.
  • Synced breaks when everyone keeps the session open.
