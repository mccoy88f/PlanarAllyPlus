Scheda del personaggio

Scheda D&D 5e (layout italiano v2) con campi collegati, salvataggio in campagna e collegamento ai segnalini.

Elenco e selezione

Nella barra laterale trovi le schede della campagna. Selezionane una per modificarla; senza selezione la modifica non è disponibile.

Collegamento a personaggio e token

Puoi associare una scheda a un personaggio della campagna e, dal menu contestuale del segnalino, aprire rapidamente la scheda collegata.

Compendium

Nei campi testo supportati, i collegamenti al compendio (quinta edizione) permettono di inserire riferimenti a incantesimi e regole coerenti con i tuoi contenuti.

Import / Export

Esporta la scheda in JSON per backup o importala da file (utile per migrazioni o creazione rapida di PNG).

Permessi

Il Master vede e modifica tutte le schede secondo le regole della campagna; i giocatori lavorano sulle proprie schede (e su quelle rese visibili).

Character Sheet

D&D 5e-style sheet with linked fields, campaign save, and token integration.

List and selection

Use the sidebar to pick a sheet. Editing actions apply only when a sheet is selected.

Character and token link

Associate a sheet with a campaign character; open it from the token context menu when linked.

Compendium

Supported text fields accept compendium links so spells and rules stay consistent with your installed content.

Import / Export

Export to JSON for backup or import from a file for migrations or quick PC creation.

Permissions

The GM manages visibility and edits per campaign rules; players edit their own sheets (and shared ones as allowed).
