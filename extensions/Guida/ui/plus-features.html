Guida alle Estensioni PlanarAlly Plus

Scopri l'ecosistema di estensioni e le funzioni esclusive incluse nel fork PlanarAlly Plus. Queste risorse aggiungono strumenti potenti per Master e Giocatori.

Disclaimer: Le estensioni sono sviluppate dalla community o integrate da repository esterne. Usale con consapevolezza. Assicurati di abilitare le estensioni nelle Impostazioni Account per poterle installare.

Estensioni Integrate

MapsGen Maps

Generatore procedurale di dungeon ed edifici. Crea mappe complete in pochi secondi.

  • Modalità Dungeon: archetipi (cripta, tempio, tana), simmetria e complessità variabile.
  • Modalità Edifici: layout per case, taverne, locande con stanze e arredi.
  • AI Realistic: trasforma le mappe schematiche in grafiche realistiche.

Scheda del personaggio System

Gestione avanzata delle schede personaggio D&D 5e e generiche.

  • Collega le schede ai segnalini (token) sulla mappa.
  • Integrazione con il Compendium per aggiungere incantesimi e abilità.
  • Supporto Import/Export in formato JSON.

Compendium Rules

Database di regole, mostri e incantesimi sempre a portata di mano.

  • Installa compendi SRD 5e o database personalizzati (es. quintaedizione.online).
  • Condividi voci direttamente nella chat per i giocatori.
  • Supporto Deep Linking: [testo](qe:...) nelle note.

Musica d'ambiente Audio

Controlla l'atmosfera sonora della tua sessione.

  • Crea e salva playlist direttamente dagli asset della campagna.
  • Sincronizzazione in tempo reale del brano e del volume per tutti i giocatori.
  • Modalità Loop e riordino brani tramite drag & drop.

Documenti Utility

Visualizzatore avanzato di documenti PDF in-game.

  • Pagine sfogliabili con anteprima rapida.
  • Estrazione testo per ricerche veloci all'interno dei manuali.
  • Condivisione di link a pagine specifiche nella chat di gioco.

Watabou Generators Maps

Integrazione con i celebri generatori di Watabou.

  • Genera istantaneamente Città, Villaggi, Regioni e Dungeon.
  • Wrapper ottimizzato per l'uso all'interno del VTT.
  • Ideale per improvvisare location durante la sessione.

AI Generator AI

L'intelligenza artificiale integrata per assistere il DM.

  • Generazione di BG, descrizioni e dialoghi tramite OpenRouter.
  • Creazione di immagini per nuovi asset e segnalini.
  • Potenzia MapsGen con la trasformazione in mappe realistiche.

Gestione del tempo Tool

Gestisci il tempo e i turni con precisione.

  • Timer e conti alla rovescia configurabili.
  • Esecuzione in background: continua a scorrere anche se il modale è chiuso.
  • Avviso audio al termine del tempo.

Assets Installer Setup

Installa pacchetti massivi di risorse in un lampo.

  • Carica file ZIP per popolare automaticamente la libreria asset.
  • Gestione pacchetti: installa bundle di mostri, mappe o icone.

Funzioni Esclusive Plus

PWA (Progressive Web App)

Usa PlanarAlly come un'app nativa. Supporto per installazione su Desktop, Android e iOS con icona dedicata e prestazioni ottimizzate.

Plus Launcher

Dimentica il terminale. Un eseguibile dedicato per avviare, configurare e aggiornare il server con un'interfaccia semplice.

Unified Assets System

Gestione file basata su database SQL per anteprime asincrone istantanee, supporto a nomi duplicati e organizzazione in cartelle virtuali.

Traduzione Completa

Supporto nativo e curato per Italiano e Inglese in ogni parte del software, incluse tutte le estensioni qui elencate.

Installazione di estensioni

  1. Vai al Cruscotto e apri la sezione Estensioni.
  2. Clicca su Aggiungi estensione.
  3. Scegli come installare:
    • Carica file ZIP: se hai un file ZIP dell'estensione.
    • Installa da URL: se l'estensione è disponibile via URL.
Informazione: Le estensioni vengono caricate nel contesto della campagna. Assicurati di avere una campagna aperta o di averla selezionata per gestire le estensioni correttamente.

PlanarAlly Plus Extensions Guide

Discover the extension ecosystem and exclusive features included in the PlanarAlly Plus fork. These resources provide powerful tools for GMs and Players.

Disclaimer: Extensions are developed by the community or integrated from external repositories. Use them at your own risk. Make sure to enable extensions in Account Settings before installing them.

Integrated Extensions

MapsGen Maps

Procedural dungeon and building generator. Create complete maps in seconds.

  • Dungeon mode: archetypes (crypt, temple, lair), symmetry, and variable complexity.
  • Buildings mode: layouts for houses, taverns, inns with rooms and furniture.
  • AI Realistic: transforms schematic maps into high-quality realistic graphics.

Character Sheet System

Advanced management for D&D 5e and generic character sheets.

  • Link sheets to tokens on the map.
  • Integration with the Compendium to add spells and abilities.
  • JSON Import/Export support.

Compendium Rules

A rule, monster, and spell database always at your fingertips.

  • Install SRD 5e compendiums or custom databases.
  • Share entries directly into the chat for players.
  • Deep Linking support: [text](qe:...) in notes.

Ambient Music Audio

Control the sound atmosphere of your session.

  • Create and save playlists directly from campaign assets.
  • Real-time track and volume synchronization for all players.
  • Loop mode and track reordering via drag & drop.

Documents Utility

Advanced in-game PDF document viewer.

  • Browsable pages with quick preview.
  • Text extraction for fast searching within manuals.
  • Share links to specific pages in the game chat.

Watabou Generators Maps

Integration with the famous Watabou generators.

  • Instantly generate Cities, Villages, Regions, and Dungeons.
  • Optimized wrapper for VTT usage.
  • Perfect for improvising locations during sessions.

AI Generator AI

Integrated AI tool to assist the GM.

  • Generate background stories, descriptions, and dialogues via OpenRouter.
  • Create images for new assets and tokens.
  • Powers MapsGen with realistic map transformations.

Time Manager Tool

Manage time and turns with precision.

  • Configurable timers and countdowns.
  • Background execution: keeps running even when the modal is closed.
  • Audio alert when time is up.

Assets Installer Setup

Install massive asset packs in a flash.

  • Upload ZIP files to automatically populate the asset library.
  • Pack management: install bundles of monsters, maps, or icons.

Exclusive Plus Features

PWA (Progressive Web App)

Use PlanarAlly like a native app. Support for installation on Desktop, Android, and iOS with a dedicated icon and optimized performance.

Plus Launcher

Forget the terminal. A dedicated executable to start, configure, and update the server with a simple interface.

Unified Assets System

SQL database-based file management for instant asynchronous previews, duplicate name support, and virtual folder organization.

Full Localization

Native and curated support for Italian and English throughout the software, including all extensions listed here.

Installing Extensions

  1. Go to the Dashboard and open the Extensions section.
  2. Click on Add extension.
  3. Choose how to install:
    • Upload ZIP file: if you have a ZIP file of the extension.
    • Install from URL: if the extension is available via URL.
Information: Extensions are loaded within the campaign context. Make sure you have a campaign open or have it selected to manage extensions correctly.
