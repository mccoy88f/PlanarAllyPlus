Documenti (PDF)

Estensione che incorpora un visualizzatore PDF (basato su motore compatibile con PDF.js) per leggere manuali, avventure e reference direttamente nel client, senza cambiare finestra.

Apertura e navigazione

  • Seleziona un PDF dagli asset della campagna (o dal percorso previsto dall’estensione).
  • Scorri le pagine con controlli avanti/indietro; spesso è presente una griglia o elenco di miniature per saltare a una sezione.
  • Zoom e adattamento pagina dipendono dal viewer integrato — utile per testo piccolo o mappe a pagina intera.

Ricerca nel testo

Dopo l’estrazione del testo dal documento (operazione lato client/server secondo il build), puoi usare una casella di ricerca per trovare termini all’interno del PDF caricato, senza dover sfogliare manualmente centinaia di pagine.

Prestazioni e caricamento

I PDF grandi possono caricarsi in modo progressivo; l’estensione evita richieste duplicate quando apri la stessa anteprima più volte (cache lato server/client dove implementata).

Condivisione in chat

Puoi inviare un link a una pagina specifica (o un riferimento interpretato dal client) nella chat di gioco: gli altri partecipanti aprono lo stesso punto del manuale.

Casi d’uso

  • Consultare regole durante il combattimento.
  • Leggere insegne di avventure pubblicate in PDF.
  • Mostrare illustrazioni a pagina intera al gruppo.

Documents (PDF)

Embeds a PDF viewer (PDF.js–compatible stack) so you can read manuals, adventures, and references inside the client without alt-tabbing.

Opening and navigation

  • Pick a PDF from campaign assets (or the path the extension expects).
  • Page navigation with next/prev; thumbnails or a page strip when available.
  • Zoom/fit modes depend on the embedded viewer—handy for tiny stat blocks or full-page maps.

Text search

After text extraction, a search box can locate terms across the loaded document instead of manual paging.

Performance

Large PDFs may load progressively; duplicate fetch avoidance helps when reopening the same preview.

Sharing

Share a page-specific link (or client-resolved reference) to game chat so the table opens the same section.

Use cases

  • Rule lookups during combat.
  • Reading published adventure PDFs.
  • Full-page art for the table.
