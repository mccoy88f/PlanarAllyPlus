Watabou Generators

L’estensione incapsula i generatori web pubblici di Watabou (Medieval Fantasy City Generator, Village, One Page Dungeon, regioni, ecc.) in un pannello iframe adattato al layout PlanarAlly.

Cosa puoi generare

In base ai generatori inclusi nella build:

  • Città e villaggi con quartieri, strade e iconografia fantasy.
  • Mappe regionali o macro-aree per viaggi.
  • Dungeon stile one-page con stanze e collegamenti.

Ogni generatore ha i propri controlli (seed, tema, densità): usa il seed per ricreare la stessa mappa in una sessione futura.

Flusso verso il tavolo

  • Genera finché non trovi un risultato utile.
  • Esporta o acquisisci l’immagine secondo i pulsanti del generatore (PNG/SVG dove disponibile).
  • Carica il file negli asset della campagna e impostalo come sfondo mappa, oppure usalo come reference per il Master.

Perché usarlo nel VTT

Evita di saltare tra il browser esterno e PlanarAlly: tutto resta nella stessa sessione. È ideale per improvvisazione quando la festa entra in una città non preparata.

Limitazioni

I generatori dipendono dalla disponibilità dei siti Watabou; la connessione di rete deve permettere il caricamento dell’iframe. Funzioni avanzate di export sono quelle offerte dal generatore stesso.

Watabou Generators

Embeds public Watabou web tools (city, village, dungeon, regions, etc.) inside a PlanarAlly-sized iframe panel.

What you can roll

  • Urban maps with districts and roads.
  • Regional overland maps.
  • One-page dungeons with rooms and links.

Each tool exposes its own knobs (seed, palette, density); capture the seed to reuse a layout later.

Getting it to the table

  • Iterate until you like the output.
  • Export via the generator’s buttons (PNG/SVG when offered).
  • Upload to campaign assets and set as a map background or GM reference.

Why inside the VTT

Keeps focus in one window—great for improv when the party visits an unplanned settlement.

Limits

Relies on Watabou’s web apps; network access must allow the iframe. Export features are whatever the generator exposes.
