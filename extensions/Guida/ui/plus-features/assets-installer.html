Assets Installer

Installa in blocco risorse nella libreria asset della campagna caricando archivi ZIP strutturati (icone, mappe, token, ecc.).

Flusso tipico

Seleziona un pacchetto ZIP compatibile; il server estrae file e cartelle nella libreria rispettando la struttura del pacchetto.

Quando usarlo

Utile dopo aver scaricato bundle di terze parti o per migrare cartelle di asset senza caricare file uno per uno dall’interfaccia standard.

Permessi

L’operazione riguarda gli asset della campagna: servono i permessi adeguati (in genere Master o chi gestisce la libreria).

Assets Installer

Bulk-install resources into the campaign asset library by uploading structured ZIP packs (icons, maps, tokens, etc.).

Typical flow

Pick a compatible ZIP; the server unpacks files and folders into the library following the archive layout.

When to use it

Great after downloading third-party bundles or when migrating asset folders without manual per-file uploads.

Permissions

Campaign asset operations require appropriate rights (typically GM or whoever manages the library).
