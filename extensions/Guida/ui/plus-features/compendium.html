Compendium

Consulta regole, mostri, incantesimi e contenuti personalizzati senza uscire dal tavolo virtuale.

Installazione contenuti

Da cruscotto o flusso dedicato puoi importare database JSON/ZIP (es. SRD 5e o pacchetti di terze parti). Ogni installazione aggiunge compendi e collezioni navigabili dall’estensione.

Navigazione e ricerca

Browser ad albero, ricerca globale e tag aiutano a trovare una voce durante la sessione.

Condivisione in chat

Invia un estratto o un riassunto della voce direttamente in chat così tutti i giocatori vedono lo stesso testo.

Deep linking nelle note

Nelle note puoi usare sintassi del tipo [testo](qe:…) per puntare a voci del compendio, creando collegamenti cliccabili.

Immagini e asset

Le illustrazioni nelle voci possono essere aggiunte alla libreria asset della campagna (percorso organizzato sotto extensions/compendium/…) e trascinate sulla mappa come qualsiasi altro file.

Compendium

Rules, monsters, spells, and homebrew—available inside the VTT.

Installing content

Import JSON/ZIP packs (SRD 5e or third-party databases). Each install adds compendiums and collections you can browse from the extension.

Browsing and search

Tree navigation, global search, and tags help you find entries mid-session.

Share to chat

Push a snippet or summary to game chat so everyone sees the same text.

Deep linking in notes

Use [label](qe:…) syntax in notes to link to compendium entries.

Images and assets

Art in entries can be sent to the campaign asset library (under extensions/compendium/…) and dragged to the map like any other asset.
